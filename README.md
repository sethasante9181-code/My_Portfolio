# Personal Portfolio

A simple portfolio website built with HTML, CSS, and JavaScript. Use this project as a starting point to add your own name, projects, and contact details.

## Files

- `index.html` — main page markup
- `styles.css` — responsive styling and layout
- `script.js` — navigation toggle and form interaction

## Customize

1. Open `index.html` and replace placeholder text like `Your Name`, `hello@example.com`, and project descriptions.
2. Adjust colors or fonts in `styles.css`.
3. Add real project links or images as needed.

## Run locally

Open `index.html` in a browser or use a local server if you prefer. For example, with Visual Studio Code, install the Live Server extension and open the page.
